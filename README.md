# Shrine::Scp

Scp storage plugin for Shrine attachment and upload toolkit

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'shrine-scp'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install shrine-scp
